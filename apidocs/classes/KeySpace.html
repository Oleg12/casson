KeySpace Class

Defined in: lib/keyspace.js:4
Module: casson

Keyspace, like Cassandra keyspace, maintains a list of tables. It can be stored as a JSON files and loaded from it, and the application can thus maintain a simple mirror Cassandra tables in javascript.
