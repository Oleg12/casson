Add where clause

Parameters:

    map

  • [where_and] Boolean optional

    And with where

Example:

  {'col1' : '5', 'col2' : 'p'}  // col1 = 5 and col2 = 'p'
  {'col1' : [3, 5]}             // col1 IN [3, 5]
  {'col1' : {'>=' : 5}}         // col1 >= 5
  {'col1' : {'>' : 5, '<' : 9}} // col1 > 5 and col1 < 9
